# Predicting Profitability in Retail Transactions

This project uses **machine learning** to predict the **profit** of retail transactions from the **Superstore Sales dataset**.  
It follows the **CRISP-DM** process — data cleaning, feature engineering, modeling, and evaluation — to identify the main factors driving profitability (e.g. discounts, sales volume, and shipping time).
